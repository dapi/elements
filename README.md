# Tiny Custom Elements with no dependency

## Installation

```html
<script defer href="https://cdn.jsdelivr.net/gh/dapi/elements@master/spinner.js"></script>
```

## Usage

```html
<dapi-spinner></dapi-spinner>
```

More demos and examples - https://dapi.github.io/elements/

## Development

> yarn dev

Deploy to gh-pages

> yarn deploy
